Accessibility Audit

This page summarizes key accessibility notes for VELARION Skies.

Passes

  • Semantic landmarks (header, main, footer) and headings hierarchy.
  • Skip link for keyboard users.
  • Focus-visible styles on interactive elements.

Fixes

  • Ensure all images have descriptive alt text (decorative images: empty alt).
  • Verify color contrast ≥ 4.5:1 for text over imagery.
  • Add aria-labels to icon-only links or buttons.
